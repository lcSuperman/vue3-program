

# vue-program
用vue3 + @vue/cli + element-plus + vue-router +axios 写的后台管理系统框架，接口调用的是利用mock.js模拟的数据,由于目前还不太熟悉Ts，所以项目实用Js写的；

# 前端代码
### 把项目拉下来之后初始化包
```
npm install
```

### 然后输入下面命令启动前端项目
```
npm run serve
```

# 后端代码

### 进入serve的目录下
```
cd serve
```
### 初始化包
```
npm install
```
### 然后输入下面命令启动后端服务
```
npm run dev
```
### 直接在浏览器打开 http://localhost:3000 如果有数据，服务就启动成功
